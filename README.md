# HubDoBem

## Descrição
Projeto web para ONG Ruby do Bem, com funcionalidades de cadastro, navegação e interatividade.

## Tecnologias utilizadas
- HTML5
- CSS3
- JavaScript

## Funcionalidades
- SPA (Single Page Application)
- Validação de formulários
- Layout responsivo
- Integração com GitHub Pages
